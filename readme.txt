install ruby 1.9.2p180

install ruby game: https://github.com/rubygame/rubygame/wiki/Install

install gems:

	gem install rubygame 
	
	gem install log4r
	
	gem install mocha
	


to run the game, cd to the root directory and exec "ruby run.rb"

to run the unit tests, cd to the root directory  and exec "ruby tests.rb"
